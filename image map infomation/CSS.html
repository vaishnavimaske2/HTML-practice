CSS (Cascading style sheet)

CSS is used to style and design web pages.

  • HTML - structure
  • CSS - design

It controls:

  • Colors
  • Layout
  • Fonts
  • Spcaing
  • Animations

Ways to apply CSS

Inline CSS

  • Inside HTML tag
  • Not recommended for large projects

Internal CSS

  • Inside style tag
  • Used for single page

External CSS

  • Separate .css file
  • clean and reusable
  • Most used in real projects
