HTML information

HTML (HyperText Markup Language) is the Standard Language
used to create web pages.

  • It defines the structure of a webpage.
  • It tells the browser what to display

Key points about HTML

  • It is a markup language, not a programming language.
  • It uses tags to define elements.
  • It works with:
    • CSS (for desgin)
    • JavaScript (for functionality)

How HTML works

  1. You write HTML code
  2. Browser reads it
  3. Converts it into visible webpage

browser creates DOM (Document Object Model).

Advantages of HTML

  • Easy to learn
  • Platform independent
  • Supported by all browsers
  • Free abd widely used

Limitation of HTML

  • Cannot create dynamic content alone
  • Needs CSS and JavaScript
  • No logic or conditions
